## Запуск приложения

Для запуска приложения выполните команду:

```bash
pnpm run dev
```

## Переменные окружения

Перед запуском приложения необходимо указать следующие переменные в `.env` файле:

```env
ENV=

CLIENT_ORIGIN=
DISCORD_CLIENT_ID=
DISCORD_CLIENT_SECRET=

BETTER_AUTH_SECRET=
BETTER_AUTH_URL=
```
